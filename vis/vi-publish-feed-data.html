Publish Feed Data.vi

18 Jun 2014

Publishes analog waveform data to the Cloud Dashboard where is it available in the feeds view. If no feed name is specified or if the feed is not found an error is generated.

If the Create feed if not found? input is TRUE and the feed does not already exist the VI will attempt to create the feed. If this input if TRUE and the feed does exist, data is published to the feed and this input is ignored.

This VI is not high throughput and should be called sparingly.
